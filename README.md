# Superstore Sales Performance Analysis

## Project Overview
Analyzed 51,290 sales records of a global retail store 
to identify key revenue and profit drivers.

## Tools Used
- Microsoft Excel (Data Cleaning & Analysis)
- Power BI (Interactive Dashboard)

## Key Insights
- Central region leads with $2.8M in sales
- Tables sub-category losing $64K despite $757K revenue
- Technology is most profitable category (14% margin)
- Consumer segment contributes 51.5% of total revenue
- Sales doubled from $2.26M (2011) to $4.30M (2014)
- Tamara Chand is most profitable customer
